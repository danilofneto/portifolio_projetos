    Health Insurance Cross-Sell

    Neste projeto, usei técnicas de classificação para classificar na base de clientes de uma companhia de seguros quais são os mais propensos a assinar um seguro de veículo. Os resultados obtidos em nosso modelo de machine learning são disponibilizados para o setor de call center a partir de uma API conectada ao Google Sheets que realiza o ranking dos clientes em tempo real. Simulando 20.000 leads, o modelo proposto é cerca de 2,5 vezes melhor do que a escolha aleatória, gerando um retorno de US$ 2,9 milhões para a empresa.

    As ferramentas utilizadas foram:

    • Git, Gitlab e Github.
    • Python, Pandas, Matplotlib e Seaborn.
    • Jupyter Notebook e Visual Studio Code.
